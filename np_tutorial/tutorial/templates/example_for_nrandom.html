Пример

Условие задачи "Сборы на прогулку"

Нужно принять решение об экипировке для поездки на природу.

Есть такие варианты:

\(\alpha_{1}\) - одеться как можно легко

\(\alpha_{2}\) - одеться легко, но взять зонт

\(\alpha_{3}\) - одеться как можно теплее

Возможные состояния природы

\(\beta_{1}\) - солнечная погода

\(\beta_{2}\) - проливной дождь

Продолжая общую теорию. Пусть Задана матрица потерь L.

Предположим \(\beta_{1}\) - контролируемое состояние.

\(\beta_{1}\) \(\beta_{2}\)
\(\alpha_{1}\) 0 1
\(\alpha_{2}\) 0,1 0,9
\(\alpha_{3}\) 0,6 0,4

\(L^{*}_{пороговое}\) = 0,5

Вычеркиваем сроки, у которых в столбце \(\beta_{1}\) значения больше порогового.

\(\beta_{1}\) \(\beta_{2}\)
\(\alpha_{1}\) 0 1
\(\alpha_{2}\) 0.1 0.9
\(\alpha_{3}\) 0.6 0.4

То есть последнее решение нам не подходит. Далее выбираем то решение, которое при неконтролируемом состоянии природы приносит наименьшие потери. В данном примере по столбцу \(\beta_{2}\) наименьшие потери 0.9. То есть нам подходит решение \(\alpha_{2}\)
